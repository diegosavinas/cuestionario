## Descripción

Este proyecto es un punto de partida para crear un cuestionario de accesibilidad HTML/CSS. Utiliza etiquetas semánticas y atributos ARIA para mejorar la accesibilidad de la página. Los estilos CSS se aplican para mejorar la apariencia visual de la página y proporcionar una experiencia de usuario agradable.

## Uso

Puedes usar este proyecto como base para crear tu propio cuestionario de accesibilidad o como ejemplo de cómo estructurar una página web semántica y accesible. También puedes personalizar los estilos CSS según tus necesidades específicas de diseño.
